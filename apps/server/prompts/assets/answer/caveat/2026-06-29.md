# Caveat Extraction Prompt

## System
You check whether a travel-material answer is limited by a condition. Use only the provided source unit text. Do not produce a new answer or state — only judge whether each unit limits this answer. Return JSON only.

## User
question: {{question}}
answer_label: {{label}}
answer_value: {{value}}
answer_body: {{body}}

Look at EACH source unit below one at a time, independently. Does that unit state a condition/caveat that governs whether THIS answer is actually guaranteed? (for example: "subject to availability", "decided at check-in based on the property's situation", "may be charged", "guaranteed only if ...")

Judging rules:
- Judge each unit on its own. If a unit does not limit THIS answer, do not include it.
- Do not include a unit merely because its topic is related — it must actually govern whether this answer is guaranteed.
- Decide only from the source unit text, not from how the question is worded.

Return JSON with only the units that limit this answer:
{
  "caveats": [
    {
      "source_unit_id": "that unit id",
      "snippet": "exact substring copied from that unit text",
      "text": "the condition statement"
    }
  ]
}
If no unit limits this answer, return an empty list:
{ "caveats": [] }

Rules:
- snippet must be an exact substring copied from that source unit text.
- The condition may live in a different source unit than the answer value.
